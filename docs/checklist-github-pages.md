# Checklist de Publicacao — GitHub Pages

## Antes de publicar

- [ ] Repo movido para fora do vault
- [ ] `main` como branch padrao
- [ ] Workflow `deploy-gh-pages.yml` presente
- [ ] Docs revisados em `docs/`

## Publicacao

- [ ] Fazer commit e push
- [ ] Verificar workflow rodando no GitHub Actions
- [ ] Ativar GitHub Pages (Settings → Pages)
- [ ] Definir source como `gh-pages`

## Depois de publicar

- [ ] Testar links principais
- [ ] Validar navegacao do `docs/index.md`
- [ ] Corrigir links quebrados
