# Visao Geral do Treinamento

O treinamento de telemetria da Serra Rocketry prepara novos membros para atuar nos projetos de Flight Computer e Helike.

## O que voce vai aprender

- Fundamentos de telemetria aplicada
- Python para parsing e logging
- Arduino IDE e PlatformIO
- KiCad basico
- Fluxo de trabalho no GitHub

## Como funciona

- 8 semanas de entregas
- Issues semanais com criterios claros
- PRs revisados toda semana
