# Guia do Instrutor

Este guia descreve como conduzir o treinamento semana a semana.

## Sessao semanal de revisao de PR

**Objetivo**: garantir alinhamento tecnico e boas praticas.

**Roteiro sugerido**:

1. Cada membro apresenta o PR (2–3 min)
2. Revisor aponta 1–2 melhorias objetivas
3. Confirmar criterios de aceite
4. Encerrar com proxima entrega

## Boas praticas de revisao

- Seja objetivo e gentil
- Evite refatoracoes grandes fora do escopo
- Priorize clareza, legibilidade e reproducibilidade
- Incentive documentacao minima (README)

## Checklist rapido do revisor

- [ ] PR segue o template
- [ ] Entrega atende o objetivo da semana
- [ ] README explica como rodar/testar
- [ ] Nao ha dependencias ocultas

## Padrao para issues semanais

- Uma issue por tarefa principal
- Critérios claros e testaveis
- Link para trilha correspondente
