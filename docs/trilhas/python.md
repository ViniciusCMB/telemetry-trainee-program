# Trilha Python

## Objetivo

Dar base de Python aplicada a telemetria: leitura, parsing, validação e logging.

## Competencias esperadas

- Ler e limpar dados de telemetria
- Parsear pacotes simples
- Validar dados e registrar logs

## Pre-requisitos

- Python 3.10+
- VSCode (ou editor equivalente)

## Conteudos

1. **Leitura de dados** (CSV/JSON)
2. **Parsing de pacotes** (string → dicionario)
3. **Validacao simples** (tamanho/checksum)
4. **Persistencia** (log em arquivo)

## Exercicios sugeridos

1. Ler CSV e calcular media/variancia
2. Criar parser de pacote com delimitadores
3. Implementar validacao simples (tamanho/checksum)
4. Gerar log em arquivo com timestamp

## Proximo nivel (opcional)

- Detectar apogeu simples a partir de serie de altitude

## Entregas relacionadas

- Semana 1: Script CSV + resumo
- Semana 2: Parser + validacao

## Criterios de aceite (base)

- Codigo executa sem erros
- Output legivel e documentado
- README com instrucoes
