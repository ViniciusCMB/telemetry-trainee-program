# Trilha Arduino IDE

## Objetivo

Introduzir fundamentos de microcontroladores e comunicacao serial usando Arduino IDE.

## Competencias esperadas

- Criar sketch funcional
- Ler sensor (ou simular leitura)
- Enviar pacote pela serial

## Pre-requisitos

- Arduino IDE instalado
- Placa compativel (ex: Arduino/ESP)

## Conteudos

1. **GPIO** (blink)
2. **Leitura de sensor** (simulado)
3. **Serial** (envio de pacote formatado)

## Exercicios sugeridos

1. Blink com controle de tempo
2. Simular leitura de sensor com `analogRead`
3. Enviar pacote serial com campos fixos

## Proximo nivel (opcional)

- Garantir taxa de amostragem constante (ex: 20 Hz)

## Entregas relacionadas

- Semana 3: Blink + sensor + serial

## Criterios de aceite (base)

- Codigo compila e faz upload
- Serial envia pacote no formato definido
