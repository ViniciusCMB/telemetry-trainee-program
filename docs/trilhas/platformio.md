# Trilha PlatformIO

## Objetivo

Migrar o projeto Arduino para uma estrutura organizada e escalavel em PlatformIO.

## Competencias esperadas

- Estruturar projeto com `src/` e `lib/`
- Configurar `platformio.ini`
- Compilar e fazer upload

## Pre-requisitos

- VSCode + PlatformIO instalado

## Conteudos

1. **Estrutura de projeto** (src/, include/, lib/)
2. **Bibliotecas** (gerenciamento via platformio.ini)
3. **Build e upload**

## Exercicios sugeridos

1. Migrar sketch do Arduino IDE
2. Separar funcoes em arquivos
3. Adicionar biblioteca via `platformio.ini`
