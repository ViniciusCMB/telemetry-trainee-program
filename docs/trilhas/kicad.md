# Trilha KiCad

## Objetivo

Ensinar o basico de esquematico e checklist ERC para projetos simples.

## Competencias esperadas

- Criar esquematico simples
- Rodar ERC sem erros criticos
- Exportar PDF

## Pre-requisitos

- KiCad instalado

## Conteudos

1. **Schematic** (sensor + MCU)
2. **ERC** (verificacao de regras)
3. **Exportacao** (PDF do esquematico)

## Exercicios sugeridos

1. Montar esquematico MCU + sensor
2. Corrigir erros de ERC
3. Exportar PDF final

## Proximo nivel (opcional)

- Checklist eletrico simples (GND/VCC, desacoplamento)

## Entregas relacionadas

- Semana 6: Esquematico + ERC

## Criterios de aceite (base)

- ERC sem erros criticos
- PDF exportado
