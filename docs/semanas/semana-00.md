# Semana 0 — Onboarding & GitHub

## Objetivo

Alinhar ferramentas e fluxo de trabalho com issues e PRs.

## Entregas (issues)

- Setup do ambiente (VSCode, Python, Arduino, PlatformIO, KiCad)
- Clonar repo + criar branch `nome-sobrenome`
- PR “hello world” + README pessoal

## Criterios de aceite

- PR segue template
- Branch nomeada corretamente
- README com instrucoes basicas
