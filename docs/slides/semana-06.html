  • Exportar PDF e Gerber para fabricação

Visão Geral do KiCad

KiCad é um suite EDA open-source com 5 ferramentas principais:

  • Eeschema — editor de esquemáticos
  • Pcbnew — editor de PCB layout
  • Symbol Editor — criar/editar símbolos
  • Footprint Editor — criar/editar footprints
  • GerbView — visualizar Gerbers

Fluxo: Esquemático → Anotação → Associação de footprints → PCB → Gerber

Criando um Projeto

projeto-telemetria/
├── telemetria.kicad_pro    # Arquivo de projeto
├── telemetria.kicad_sch    # Esquematico
├── telemetria.kicad_pcb    # PCB Layout
├── symbols/                # Simbolos personalizados
├── footprints/             # Footprints personalizados
└── fabrication/            # Saida Gerber/PDF

No KiCad: File → New → Project → escolher pasta.

Eeschema — Esquemático MCU + Sensor

Componentes principais de um nó de telemetria:

  • Microcontrolador (ATmega328P, RP2040, STM32)
  • Sensor BMP280 (I2C)
  • Conector USB (alimentação + dados)
  • Regulador de tensão 3.3V
  • Capacitores de desacoplamento
  • Resistores de pull-up I2C
  • Conector para bateria LiPo

Adicionando Componentes

Tecla de atalho: 'A' (Add Symbol)

Buscar e colocar:
- ATmega328P_PDIP        (MCU)
- BMP280                  (sensor pressao)
- USB_B_Micro             (conector USB)
- LM1117-3.3              (regulador)
- C_Small (100nF, 10uF)  (capacitores)
- R_Small (4k7)          (pull-ups I2C)
- CONN_01x02              (bateria)

Use fios (wire) para conectar os pinos.

Exemplo de Conexões I2C

                       3.3V
                        │
                    ┌───┤
                    │   │
                  4k7  4k7
                    │   │
                    │   └──────────┐
                    │              │
  MCU               │              │     BMP280
┌──────┐          ┌─┴─┐         ┌─┴──┐
│ SDA  │──────────┤   ├─────────┤ SDA│
│ SCL  │──────────┤   ├─────────┤ SCL│
│ VCC  │───3.3V   │   │         │ VCC│───3.3V
│ GND  │───GND    │   │         │ GND│───GND
└──────┘          └───┘         └────┘

I2C precisa de pull-ups (4k7) para SDA e SCL.

ERC — Electrical Rules Check

Antes de passar para PCB, verifique o esquemático:

  • Menu: Inspect → Electrical Rules Check
  • Verifica conexões soltas, pinos não conectados, conflitos
  • Corrija todos os erros (bandeiras vermelhas)
  • Avisos (amarelos) podem ser aceitos se justificados
Exemplo de erros comuns:
- Erro: Pino VCC do BMP280 nao conectado
- Erro: Saida SDA em curto com GND
- Aviso: Resistores de pull-up ausentes no I2C

ERC zero erros antes de ir para o layout!

Associando Footprints

Cada símbolo precisa de um footprint (forma física no PCB):

Menu: Tools → Assign Footprints

Exemplos:
- ATmega328P → DIP-28_W7.62mm
- BMP280 → LGA-8_2x2.5mm_P0.65mm
- 100nF → C_0805
- 4k7 → R_0805
- USB → USB_Micro-B_GCT
- Regulador → SOT-223

Use o filtro de busca para encontrar o footprint correto.

Pcbnew — Layout Básico

Princípios de um bom layout:

  • Capacitores de desacoplamento o mais próximo possível do pino VCC
  • Planos de GND contínuos (evitar rasgos)
  • I2C tracks curtas e paralelas (SDA + SCL)
  • Separar parte analógica da digital
  • Evitar ângulos de 90 nos traces (usar 45)
  • Largura do trace: sinal 0.25mm, power 0.5mm+

Roteamento Básico

Teclas uteis no Pcbnew:
'X' — iniciar roteamento (add track)
'V' — via
'R' — rotacionar componente
'F' — flip para lado oposto
'M' — mover
'Delete' — remover track

Camadas tipicas:
F.Cu (Top) — componentes e sinais
B.Cu (Bottom) — GND plane (ou sinais)
Edge.Cuts — contorno da placa

Plano de GND e Power

Use zonas de cobre para alimentação e GND:

1. Selecione a camada B.Cu
2. Menu: Place → Zone (ou 'Ctrl+Z')
3. Clique no contorno da placa
4. Net: GND
5. Preferencias: Thermal relief = none
6. Clique em 'Fill All Zones' (B)

Repita para 3.3V na camada F.Cu (se houver espaço)

Zonas de GND reduzem ruído e melhoram a integridade do sinal.

DRC — Design Rules Check

Verifica se o layout respeita as regras de fabricação:

Menu: Inspect → Design Rules Check

Regras comuns (default):
- Clearance: 0.25mm (min entre trilhas)
- Track width: 0.25mm (min)
- Via size: 0.6mm (min)
- Via drill: 0.3mm (min)
- Anular ring: 0.15mm (min)

Ajuste em: File → Board Setup → Design Rules

DRC zero erros antes de exportar!

Exportação — PDF e Gerber

PDF do esquemático:

Eeschema → File → Plot → Formato PDF → Plotar

Gerber (fabricação):

Pcbnew → File → Fabrication → Gerbers

Camadas necessarias:
- F.Cu, B.Cu (cobre)
- F.Mask, B.Mask (mascara solda)
- F.Silkscreen, B.Silkscreen (serigrafia)
- Edge.Cuts (contorno)
- Drill (.drl file)

Envie o .zip dos Gerbers para JLCPCB, PCBWay, etc.
